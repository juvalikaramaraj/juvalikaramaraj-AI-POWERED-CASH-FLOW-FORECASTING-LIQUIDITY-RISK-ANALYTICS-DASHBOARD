# 📊 Cash Flow Forecasting Dashboard

> A Financial Planning & Analysis (FP&A) project built using **Microsoft Excel** and **Power BI** to forecast cash flows, monitor liquidity, and support better financial decision-making.

---

## 📖 Project Overview

The **Cash Flow Forecasting Dashboard** is designed to help businesses analyze historical cash flow, forecast future cash positions, and evaluate liquidity under different business scenarios.

This project demonstrates practical financial modeling skills by combining Excel-based forecasting with interactive Power BI dashboards.

---

## 🎯 Project Objectives

* Forecast monthly cash inflows and outflows
* Monitor opening and closing cash balances
* Identify potential liquidity risks
* Compare Best, Base, and Worst Case scenarios
* Support budgeting and financial planning decisions
